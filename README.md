 **Optimal Trade Execution using Reinforcement Learning**



This repository contains the implementation developed for my Master's thesis.



The optimal execution of financial transactions involves determining how to buy or sell a large quantity of assets while minimizing the impact of that transaction on the market, as it can cause adverse price fluctuations. This is a significant problem in financial markets, which are complex environments and very difficult to model. The project studies the optimal execution problem in financial markets using Reinforcement Learning. 



The implementation includes:



\- PPO (Proximal Policy Optimization)

\- LSTM neural network

\- LOBSTER limit order book data available on the website: https://php.lobsterdata.com/info/DataSamples.php

\- Comparison with the Almgren-Chriss (AC) model with a risk aversion level of 9

\- Comparison with Time Weight Average Price (TWAP) which is an Almgren-Chriss model with zero risk aversion.
