\# Optimal Trade Execution using Reinforcement Learning



This repository contains the implementation developed for my Master's thesis.



The project studies the optimal execution problem in financial markets using Reinforcement Learning.



The implementation includes:



\- PPO (Proximal Policy Optimization)

\- LSTM neural network

\- LOBSTER limit order book data

\- Comparison with Time Weight Average Price (TWAP)

\- Comparison with Almgren-Chriss
